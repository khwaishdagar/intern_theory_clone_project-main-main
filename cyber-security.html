Security Posture

Cyber Security Advisory

We adopt layered cyber defense aligned with CERT-In advisories, MeitY guidelines, and global best practices to keep learner and employer data safe.

CISO: Lt Cdr (Retd.) A. Menon Last VAPT: Oct 2025

Defense in Depth

  • Zero Trust network segmentation with MFA on every admin interface.
  • SIEM-backed monitoring and automated alert triage.
  • Daily backups stored on isolated, immutable storage.
  • Bug bounty program for responsible disclosures.

Compliance & Audits

  • ISO 27001 and SOC 2 Type II controls mapped to MSDE SOP.
  • Quarterly VAPT by CERT-In empanelled auditors.
  • Source-code reviews for critical services.
  • Secure SDLC with dependency scanning and SAST/DAST automation.

Incident Response

  • 24x7 war-room with runbooks for data breach, ransomware, or DDOS.
  • Coordination with CERT-In, NIC, and hosting partners.
  • Mandatory notification to affected users within 24 hours.
  • Post-incident RCA shared in transparency portal.

Security for Stakeholders

Administrators and partners receive role-based controls, hardware token support, and passwordless options to reduce credential theft.

  • Geo-fencing for admin logins.
  • Just-in-time privilege escalation with approval trail.
  • Audit-ready logs exported to SSDMs and MSDE.
